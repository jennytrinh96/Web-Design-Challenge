The purpose of this project was to aassemble and analyze over 500 cities using data pulled from OpenWeatherMap API. The next task was to create an analysis on how weather is affected when nearing the equator.


This project challenges us to implement skills such as Python and Pandas to create loops for API interactions to gather data on cities and construct a Pandas dataframe. Matplotlib was also used in order to plot various aspects of the weather vs. latitude. Factors such as temperature, humidity, max temperature, and cloudiness were analyzed. This site provides source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends observed.
